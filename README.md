## How to Run

Clone the repository:

git clone https://github.com/shabaabhanaan/JavaScript-Lab-01.git

Open the folder in Visual Studio Code

Navigate to any folder (A, B, C, etc.)

Right-click the HTML file

Open with Live Server or Default Browser

Press F12 to view console output if required

## Notes

Each folder represents one lab question

Each HTML file runs independently

Outputs are displayed either on the web page or in the browser console

This repository is structured for easy evaluation and lab submission
